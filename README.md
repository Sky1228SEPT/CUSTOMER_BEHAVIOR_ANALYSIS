# CUSTOMER_BEHAVIOR_ANALYSIS
Customer Behavior Analysis project using Python, MySQL, and Power BI. Includes data cleaning, exploratory data analysis (EDA), customer segmentation, SQL-based insights, and an interactive dashboard to uncover purchasing patterns, customer trends, and business opportunities for data-driven decision-making.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Examined data structure and column information.
* Verified data types and dataset dimensions.

### 2. Exploratory Data Analysis (EDA)

* Generated summary statistics.
* Identified trends, patterns, and anomalies.
* Created visualizations to understand data distribution.
* Analyzed relationships between variables.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent formats.
* Managed outliers where necessary.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Loaded cleaned data into PostgreSQL/MySQL/SQL Server.
* Wrote SQL queries to:

  * Filter and aggregate data
  * Perform joins
  * Calculate KPIs and metrics
  * Generate business insights

### 5. Dashboard Development

* Connected Power BI to the processed dataset.
* Built interactive visualizations and dashboards.
* Added filters, slicers, and KPI cards.
* Designed a user-friendly reporting interface.

### 6. Reporting & Presentation

* Documented findings in a detailed report.
* Summarized key insights and recommendations.
* Created a professional presentation using Gamma.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL/MySQL/SQL Server
* Power BI Desktop

### Installation

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Navigate to the project folder:

```bash
cd Data-Analytics-Project
```

3. Install required Python packages:

```bash
pip install pandas numpy matplotlib seaborn
```

### Execution Steps

1. Run the Python notebooks/scripts for data cleaning and EDA.
2. Import the cleaned dataset into PostgreSQL/MySQL/SQL Server.
3. Execute the SQL query files.
4. Open the Power BI dashboard (`.pbix`) file.
5. Review the generated report and presentation.
